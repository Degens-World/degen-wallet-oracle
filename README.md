# Degen Wallet Oracle

Enter any Ergo address and find out how degen you really are.

## What it does

- Fetches live balance and transaction history from the [Ergo Explorer API](https://api.ergoplatform.com)
- Scores your wallet 0–100 across four dimensions: activity, token diversity, ERG bag size, and a deterministic chaos seed
- Assigns a rank: **Paper Hands → Degen → Big Degen → Ultra Degen**
- Generates trait badges and a unique oracle prophecy for your address

## Usage

Open `index.html` in any browser — no build step, no dependencies.

Or visit the live version on [Degens.World](https://degens.world).

## Tech

Pure HTML / CSS / JS. Pulls live data from Ergo Explorer public API. Zero server-side code.
